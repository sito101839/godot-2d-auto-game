# Skills Routing

Use these project-local skills and global skills consistently.

## Project Skill

- Use `.codex/skills/godot-auto-battle-quality` for quality gates, smoke tests, β checklist work, balance simulation, save/load validation, and long-running regression checks in this project.

## GodotPrompter Skills

- Use `godot-prompter:godot-ui` and `godot-prompter:hud-system` for UI and result screens.
- Use `godot-prompter:resource-pattern` for class, trait, mission, reward, and rank data structures.
- Use `godot-prompter:save-load` for save schema or migration changes.
- Use `godot-prompter:godot-testing` for smoke tests and validation strategy.
- Use `godot-prompter:scene-organization` and `godot-prompter:component-system` before splitting `BattleManager.gd`.

## Shared Workflow Skill

- Use `godot-dev-workflow` for headless Godot checks, smoke-test pass-marker conventions, Windows crash-dialog suppression, and docs hygiene.
