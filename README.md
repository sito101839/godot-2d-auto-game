# godot-2d-auto-game

Godot 4.6 の 2D オートバトル最小プロトタイプです。

## 起動

```powershell
godot --path .
```

## 検証

```powershell
godot --headless --path . --quit
godot --headless --path . --script res://scripts/tools/hello_world_smoke_test.gd
godot --headless --path . --script res://scripts/tools/battle_smoke_test.gd
```
