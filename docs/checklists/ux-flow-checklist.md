# UX導線チェックリスト

β版でも、プレイヤーが次に何をするか迷わない状態を目標にする。

## ギルド画面

- [x] ギルド状況が最初に確認できる。
- [x] 初回表示で基本の流れが分かる。
- [x] 次にやることが専用パネルで表示される。
- [x] 重要な導線がスクロール外の上部エリアにある。
- [x] 任務ターンと大会ターンの違いが表示で分かる。
- [x] 直近の結果が編成や操作ボタンから分かれている。
- [x] 年度末/3年終了の節目レポートが専用表示される。
- [x] 3年終了後は主行動が無効化される。
- [x] メンバー一覧で出撃中/控えが分かる。
- [x] メンバー一覧で職業ごとの役割目安が分かる。
- [x] メンバー一覧が表形式で比較できる。
- [x] メンバー一覧で経験/実績が横並びで見える。
- [x] 結果表示が見出しと詳細行に分かれている。

## 行動選択

- [x] 任務開始ボタンの文言が行動として分かりやすい。
- [x] 主行動が行動選択パネルの先頭にある。
- [x] 育成メニューと保存/読込が分かれている。
- [x] 任務を経験値、Gold、名声、難度で比較できる。
- [x] 選択中の任務が分かる。
- [x] 大会ターンでは任務選択を隠し、訓練を無効化する。

## 検証

- [x] `ux_flow_smoke_test.gd` がある。
- [x] `godot --headless --path . --script res://scripts/tools/ux_flow_smoke_test.gd`
- [x] `.\scripts\tools\run_quality_checks.ps1`
- [x] `git diff --check`
